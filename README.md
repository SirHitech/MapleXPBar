# Maple XP Bar
Adds an experience bar to always track a skill on your screen. The default location is above the chatbox.

![maple-xp-bar](https://github.com/user-attachments/assets/9508b6c7-6852-4b8a-832c-9f7cceeca29b)

Many options to allow the bar to be colored, moved, resized, and anchored to other UI elements.

<img width="1547" height="1062" alt="Screenshot 2025-11-10 223233" src="https://github.com/user-attachments/assets/d3bb59b6-71bf-485e-b7b9-c637aa2c716b" />


### Modes
The default mode is Single Bar, where a skill tracked and displayed as one bar.

If you want to track 3 skills together, select Multi-Bar mode and configure the 2 additional bars in the Multi-Bar Mode section.
<img width="1534" height="1009" alt="Screenshot 2025-11-10 223954" src="https://github.com/user-attachments/assets/450f7339-fe38-4837-b98c-15ec0a62a5d6" />

Current Hitpoints and Prayer can also be tracked above your chosen skill using Health and Prayer mode.
